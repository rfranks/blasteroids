Methods

createWorld(canvas [, options])

Creates a new game World.

Parameters
Name Type Argument Description
canvas Element

the canvas element to render on
options * <optional>

options options to apply when creating the world

Properties
Name Type Argument Default Description
gravity Vector <optional>
 {x:0, y:10}

gravity for the game world, x and y component, can be horizontal, negative, etc
allowSleep boolean <optional>
 true

flag indicating whether or not entities may "sleep" when they come to rest, meaning that the entity is no longer being simulated, which can improve performance
scale number <optional>
 30

scale for rendering in the the game world (in pixels/meter or px/m)
tickFrequency number <optional>
 50

the frequency at which onTick events occur (in milliseconds, or ms)
collisionOutlines boolean <optional>
 false

flag indicating whether or not to draw outlines around entities for collision detection (useful for debugging collisions)
Source
  • public-lib/boxbox.doc.js, line 238
Returns
World

a new game world

Examples

Creating a world using the default options

var canvasElem = document.getElementById("myCanvas");
var world = boxbox.createWorld(canvasElem);

Creating a world overriding default options

var canvasElem = document.getElementById("myCanvas");
var world = boxbox.createWorld(canvasElem, {
    gravity: {x: 0, y: 20},
    scale: 60
});
